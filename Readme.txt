TC Particles is GPU particle engine for Unity. It currently supports Unity 2019.4 and up.

It was originally developed in Unity 4 for the asset store, later open sourced, and now maintained as a hobby :) 

For more info, support, documentation and the API reference, please refer to https://arthurbrussee.github.io/TCParticlesDocumentation/
